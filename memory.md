# Memory — Gryphon Group (лендінг лізингу)

Стабільні бізнес-факти про проєкт. Тільки сталі речі.

## Бізнес

- **Компанія:** Gryphon Group Sp. z o.o.
- **Адреса:** ul. Aleja Armii Krajowej 80 lok 18, 35-330 Rzeszów, Polska.
- **Продукт:** лізинг автомобілів у Польщі для іноземців та бізнесу.
- **Ключові переваги (USP):**
  - без карти побиту (для іноземців);
  - без перевірки доходів;
  - рішення за 15 хвилин;
  - авто на руках за 7 днів;
  - нові та вживані авто, знижки до 30% завдяки партнерствам з дилерами;
  - співпраця з 15+ лізинговими фірмами та банками.
- **Цільова аудиторія:** підприємці й малий бізнес; водії-фрілансери (кур'єри, таксі, логісти); IT та фахівці з доходом; українці та білоруси в Польщі.

## Бренд

- **Логотип:** золотий квадрат (#B0823C) з трьома білими «перами» крила + вордмарк «GRYPHON GROUP». Файл: `assets/logo.png` (пера прозорі — на сторінці підкладено біле тло).
- **Колір назви фірми у шапці/футері:** темно-коричневий `#4A3418` (один колір на обидва рядки).
- **Палітра:** кремова основа `#F7F2E9`, золото-бронза `#B0843B`, темні акцентні секції `#16110C`.
- **Шрифти:** заголовки — Fraunces (серіф), текст — Manrope.
- **Мови:** українська (за замовчуванням) + російська, перемикач UA/RU у шапці.

## Калькулятор лізингу

- **Ефективна річна ставка:** 10,5% (місячна ≈ 0,8355%).
- **Переплата на вартості авто за 5 років:** ~20-25% (при 10% внеску — рівно 25%, далі менше).
- **Ціна авто:** брутто. **Орієнтовна ставка:** теж брутто.
- Формула: ануїтет із залишковою вартістю (викупом). Розрахунок орієнтовний, не є офертою.

## Каталог авто (4 категорії)

1. **Легкові преміум** — виділена («featured»: золота рамка + бейдж-зірка).
2. **Легкові авто** — стандарт/комфорт-клас.
3. **Бусі для бізнесу.**
4. **Вантажівки і зестави.**

## Генерація візуалу (інструменти)

- **Higgsfield** підключений як custom MCP (план Plus). Використовуємо для фото/відео авто.
  - Модель для фото каталогу: `marketing_studio_image` (формат 3:2, 2K, ~2 кредити/кадр).
  - Застереження: AI тяжіє до реальних моделей і логотипів навіть при «no logos» — для конкретних брендів каталогу надійніше реальні фото.
- **Gemini API** підключений через ключ у `gryphon-website/.env` (`GEMINI_API_KEY`, файл у `.gitignore`).
  - Текст працює на безкоштовному тарифі.
  - **Генерація зображень (Nano Banana) потребує увімкненого billing** у проєкті Google Cloud — підписка «Pro» на застосунок Gemini цього НЕ відкриває (це різні речі). Поки не ввімкнено → лишаємось на Higgsfield.
