# Plan — Gryphon Group (лендінг лізингу)

## Поточний стан

Лендінг зібрано і працює: `gryphon-website/index.html`.
Структура секцій: шапка (sticky, UA/RU) → hero (фон — фото BMW 7) → смуга довіри → секція «Калькулятор» (окремий блок, фон — бус Mercedes) → каталог авто (4 картки) → «Для кого» (темна) → «Як це працює» (5 кроків) → «Що входить» → FAQ → фінальний CTA з формою (темна) → футер.
Динаміка: поява секцій при скролі, hover-ефекти, sticky-nav, живий калькулятор, перемикач мов із запам'ятовуванням.

## Зроблено

- Узгоджено напрямок: світла преміальна основа + темні акцентні секції.
- Інтегровано офіційний логотип (`assets/logo.png`, вирізано квадрат-крило для шапки).
- Назва фірми — один темно-коричневий колір `#4A3418`.
- Калькулятор: ефективна ставка 10,5% річних, переплата за 5 років ~20-25%; ціна та ставка — брутто.
- Каталог: 4 картки, преміум виділена; адаптив 4→2→1 без «осиротілих» карток.
- Узгоджено копірайт: скрізь «нові та вживані», знижки «до 30%».

## Наступні дії

1. **КРИТИЧНО:** підтвердити або замінити ендпоінт форми заявки (зараз заявки йдуть на старий `vibe-api…` — невідомо, чи робочий). Дати реальний приймач (CRM / Google-форма / e-mail / Telegram).
2. Вирішити: ставка нетто чи брутто (зараз обидві брутто; для B2B можлива схема «ціна брутто, рата нетто»).
3. Продовжити правки дизайну за чергою (керує користувач).
4. Опційно: точніший серіф під макет, реальні фото авто замість силуетів, деплой/хостинг.

## Як перевірити результат

- Відкрити `gryphon-website/index.html` подвійним кліком (або прев'ю-сервер `gryphon` на :8765).
- Калькулятор миттєво рахує ставку; зміна ціни/внеску/терміну/викупу оновлює результат.
- Перемикач UA/RU міняє весь текст і запам'ятовує вибір.
- Каталог: 4 картки в ряд на широкому екрані, 2×2 на середньому, 1 на мобільному.
- Форма: після відправки показує екран подяки; заявка доходить у реальний приймач (після п.1).
