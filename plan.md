# Plan — Gryphon Group (лендінг лізингу)

## Поточний стан

Лендінг зібрано, працює локально і опубліковано онлайн: `https://liliia2.github.io/gryphon-website/`.
Структура секцій: шапка (sticky, UA/RU) → hero (фон — фото BMW 7) → смуга довіри → секція «Калькулятор» (окремий блок, фон — бус Mercedes) → каталог авто (4 картки) → «Для кого» (темна) → «Як це працює» (5 кроків) → «Що входить» → FAQ → фінальний CTA з формою (темна) → футер.
Динаміка: поява секцій при скролі, hover-ефекти, sticky-nav, живий калькулятор, перемикач мов із запам'ятовуванням.
Форма заявки працює з публічного GitHub Pages: заявки йдуть у Telegram-групу і Google Sheet.

- Калькулятор миттєво рахує ставку; зміна ціни/внеску/терміну/викупу оновлює результат.
- Перемикач UA/RU міняє весь текст і запам'ятовує вибір.
- Каталог: 4 картки в ряд на широкому екрані, 2×2 на середньому, 1 на мобільному.

- Після встановлення Pixel: `PageView` і `Lead` мають бути видимі в Meta Events Manager / Pixel Helper.
